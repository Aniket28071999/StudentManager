# Student Manager (WinForms + SQLite)

This is a simple desktop application built in C# using WinForms and SQLite. It allows users to manage student data including name, email, and course.

## Features
- Add, Edit, Delete Students
- Search by Name or Course
- Save data in local SQLite database
